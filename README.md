# POLSKI--Language
Są dwie funkcje:
  -zmienna;
  -napisz;

Deklaracja zmiennej:
  zmienna {nazwa}; (zmienna o wartości wynoszącej 0)
  zmienna {nazwa} = *(np)* 1+1+{innaNazwaZmiennej}; 
Zmiana wartości Zmiennej:
  zmienna {nazwa} = *(np)* 1+1+{innaNazwaZmiennej}

Aby zmienna była stringiem trzeba na końcu wartości ustawionej dodać "":
  zmienna {nazwa} = {kody ascii decymalny}""; *przykładowo można zmienna xy = 107010"";
  107 = 'k'
  010 = '\n'*
